Blockchain Technology

  • A distributed ledger to keep track of transactions.

  • Available to anyone to see ( a way to verify the data ).

  • Everyone participating in blockchain can maintain the ledger and all will have the copy of this ledger.

  • Once changes is done in ledger, will be distributed to all the users.

Use

We can use this technology in anything we will require to keep track of transactions which could be : - Payment Processing - Digital Identification - Supply Chain Monitoring - Digital Voting

Procedure

Let's take an example of banking transaction :

  1. Members of blockchain : Bank Branches, Sender, Receiver Say Sender sent money to Receiver, instead of sending or keeping the copy of the result of this transaction only to Sender and Receiver, all the members will be forwared this ledger so that they could keep track of this info coz they are stakeholders and members. This will help in syncronization.

  2. Members : Malpot, all government institute If A is to sell his property to B, instead of keeping record only in Malpot, the copy is sent to all the members, so that whenever government has to verify if the property belongs to which individual, it could perform within any agency, not only Malpot.

Process

  • Each ledger of transaction will be added into large group of ledger after receiving copy and will ultimately processed by the Blockchain.

  • Hashes are very important as many participents has the copy of this transaction, Hash is only the way to know if info has been modified, if it has been it will not be stored and will be thrown out of the blockchain or update it.
