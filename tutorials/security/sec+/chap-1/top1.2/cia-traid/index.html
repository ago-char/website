Confidentiality

  • Prevent info from unauthorized persons/systems.

  • Implement Encryption, Access Controls (limiting access), 2FA, etc.

Integrity

  • Prevent modification of msgs at the time of transmission or in current state.

  • Any modification should be identified and caught.

  • Proof of data has not been manipulated in the middle and sender is verified.

  • Implement Hashing, Digital Signatures, Certificates, Non-repudiation, etc.

Availability

  • Authorized users should be able to access data whenever they need to do so.

  • Implement **Redundancy (build services that will always be available), Fault Tolerance (system that will continue to run even when a failure occurs), Patching (for stability and closing security holes)
