Proof of Integrity

What is sent ?

  • The Hash will determine that the data which is sent is actually the same as of in the sender side.

  • Hash is a short string of the text associate with the file sent.

  • Hash can only be changed if file itself has been modified. Even if you modify 1 single bit here and there, Hash will look completely different than of previous one. This won't tell you who sent the data, however.

Proof of Origin

Who sent ?

  • Sign the data with Private Key, which only the sender has access to.

  • Verify with the Public Key i.e. the key pair of that Private Key

As data what is sent is matched by hash and your sign is match by private-public key, there is no way to say that you have not send the data.

Being real, this process of attaching the DIGITAL SIGNATURE is often done automatically by the type of service you are using or may be your are required to check the box to add a level of intregity.
