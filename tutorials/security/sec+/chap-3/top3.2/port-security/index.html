Security on Individual Interfaces for your Devices in the Network

  • Port Security refers to the security of individual interfaces of the devices you use or the devices that are in your network.

  • When you are trying to connect to any wired or wireless network, if you are prompted for username and password, that is also one form of port security.

Extensible Authentication Protocol (EAP)

  • This is the protocol or technology behind all the authentication process happening between you and device you are trying to connect to.

  • It is not confined to one kind of device, it is developed to work on wired or wireless both systems.

  • Manufactures can develop their own EAP methods based on this protocol and RFC.

  • EAP integrates with IEEE 802.1X which is industry standard.

IEEE 802.1X

  • Sometimes you may heard that it be called as Port-based Network Access Control (NAC).

  • If you were to plug in on available switch to be on the network, you will not be able to access the network until you are authenticated with NAC, which simply means you need username and password, and this database is managed by NAC.

  • EAP integrates with NAC, also often they are used in conjunction with other auth databases like : RADIUS, LDAP, Kerberos, TACACS+, etc.

Authentication Parties

  • Supplicant - The client

  • Authenticator - The device that provides access

  • Authentication Server - Validates the client credentials
