Mobile Devices Security

  • Challenging to secure.

  • Relatively small and could be hidden anywhere.

  • Frequent connection to internet means more large attack vector.

  • Packed with personal and organizational sensitive data.

Jailbreaking/Rooting

  • These mobile devices tends to have a lot of security policies built in to their system.

  • One way of circumventing this is to jainbreak or root the system.

  • We don't commonly have access to these devices at kernal or OS level.

  • There is a complex OS running under the hood and there had been instances where attackers were able to gain access to OS level and perform anything they want.

  • You can even replace with your own OS if you manage to have jailbreaking/rooting.

  • They actually replace firmware of the system with the third party software in order to jailbreak or root their device.

  • This is done actually to install more additional features that won't be found normally.

  • All of the security policies placed by Mobile Device Manager will be circumvented.

Sideloading

  • Normally organization or OS itself will restrict you to download only from Global or local app store.

  • But if you happen to jailbreak/root, the device you can download from anywhere and hence referred as Sideloading, and there could be malicious software.

  • Again your MDM will be relatively useless.
