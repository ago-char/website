Malicious Updates

  • Verify the source

  • Check the Hash

  • Download from developer's site directly

  • Enabe signature check in OS (well enabled by default but don't disable), You may have seen "This is app from Microsoft" or "This is app from Unknwon vendor" if you have not disabled this feature

  • Avoid popup updates or do extra research on such popups:

Trusted Process of Update

  • Your trusted process will be performing updates automatically under the hood.

  • It verifies the source and update from real developer site.

  • Sometimes this automated process could download malicious software if :

    • There is malicious software in vendor site
    • This may happen if vendor itself has been attacked
    • This may also happen if vendor update chain has been compromised (someone on the update chain is threat actor and push malicious update)

  • You may want to disable this automated update but this will put you in over headache of manually verifying and downloading and updating.
