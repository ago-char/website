Operating System Vulnerabilities

  • OS itself could have vulnerabilities.

  • This is normal as they consist of millions of lines of code.

  • Microsoft releases patches every 2nd tuesday of month known as patch tuesday.

  • When these are realease, security department research and test and ultimately approve for deployment.

Should you update ?

  • YES

  • If you are using home pc, just perform backup and update.

  • If you are in company, you first go through testing process and then ultimately backup and update.

  • There is always race between you and attackers and the update window.

  • If attacker is able to write and deploy code against those vulnerabilities before you have been patch, you will likely be attacked.

  • If update requires reboot, make sure to save all data.
