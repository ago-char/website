Unsecure Public Info

  • Don't post everything online.

  • Open (not protected with key or password) repos in web holding sensitive data could leak tons of info in attacker's reconnience.

Unsecure Admin Accounts

  • Admins keeping passwordless login to root account.

  • Guessable passwords.

  • (Disable direct login, use elevated options such as su or sudo or Run as Admin).

  • Ideally you should have very few account with admin access.

Insecure Protocols

  • All traffic are in clear text.

  • Telnet, FTP, SMTP, IMAP, POP3.

  • Perform packet capture to see if you are using encrypted protocol or not.

Default Settings

  • Every system will have default login when you first enter into them.

  • Most system will ask you to change the password, Change them.

  • Mirai Botnet could find these IOT devices.

Insecure Ports

  • You may open port for some service on system.

  • Limit access to localhost or the hosts which you are intend to.

  • Use port based firewalls.

  • Open ports only on encrypted protocol.

  • Use strong password or key based auth.

  • Perform periodic audit and check.
