EndPoint Protection

  • We are using a lot of end devices through our work day (phone, pc, mac etc.)

  • So, we have to use multi-faced security

  • Use EDR (Endpoint Detection and Response)

    • Next Generation Anti-Malware Protection

    • Identify and remove known malware signatures

    • Behavioral Analysis to remove threat

      • Watch what user is doing
      • If something unusual happens report to management console

    • Machine Learning

      • Use of machine learning to train by feeding those behavioral data

    • Process Monitoring

      • Monitor every process and seek for any unusual activity

    • EDR can perfrom Root Cause Analysis

      • If it finds something not usual it does additional study to identify the root cause, and if something seems malicious report

    • Respond to threat

      • Addition to reporting, it could qurantine the system, remove the threat and even rollback to a previous known good configuration

      • This responding is mostly automated in app itself through API calls, so no need of manual inspection and even no technician required

  • Host Based Firewall

    • iptables, ufw

    • Windows firewall, windows defender

  • Host Based Intrusion Prevention System (HIPS)

    • It works on the basis of signature (stored on IPS), heuristics, behavioral changes.

    • Can identify buffer overflow, registery updates, writing to C:/Windows because it has access to host.

    • Can Access non encrypted data to analysis

  • Open Ports

  • Default Password
