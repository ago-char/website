Spraying Attack

  • Trying multiple passwords will ultimately lock out account.

  • So, instead of trying every combination, attacker try with few (depending upon lockout attempts) permissible passwords.

  • If not found, move to next account or wait till timeout.

  • Capture the Hash and perform offline attack if possible.

  • Capture the part of authentication process and try to compare by generating identical capture :

    • for eg: if a .pcap file contain some information i.e used for authentication and with all password brute forcing, if you are able to create identical scenerio i.e. the authentication part (which matters), that is the password you are looking for
