ZIP File Operations Guide

Create ZIP File

Create a ZIP file with an empty directory:

zip new.zip test/

Add contents of a directory (non-recursively) to an existing ZIP file:

zip new.zip test/*

Add contents of a directory recursively to an existing ZIP file:

zip -r new.zip test/*

Add the content of a directory including the directory itself:

zip -r new.zip test
zip -r new.zip test/

List Content

unzip -l new.zip

Append Content

Append a file or directory to an existing ZIP file:

unzip -u new.zip content

Append contents of a directory (non-recursively) to an existing ZIP file:

unzip -u new.zip content/*

Append contents of a directory recursively:

unzip -ru new.zip content

Delete Content

Delete a specific file or directory from the ZIP file:

zip -d new.zip content

Delete contents of a directory (non-recursively):

zip -d new.zip content/*

Delete contents of a directory recursively:

zip -rd new.zip content

Unzip Files

Unzip a specific file from the ZIP archive:

unzip new.zip file

Unzip a specific file to another file (using stdout , as -p is used to pipe the output):

unzip -p new.zip file > another_file
unzip -p new.zip test/dir/sub-dir/new.c > demo.c

Unzip the entire archive:

unzip new.zip

Unzip to a specified external directory:

unzip new.zip -d dir

Use with `find` and Pipes

Use `find` to create a ZIP file with files matching a pattern: (-@ is use to read files from pipe)

find playground -name "file-A" | zip -@ file-A.zip

Use `ls` and pipes to create a ZIP file from the output:(- is use to read files from pipe)

ls -l /etc/ | zip ls-etc.zip -
