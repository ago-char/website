GZIP and BZIP2 Compression Tools

GZIP

Compress files with GZIP:

$ gzip --keep --best --verbose file1
$ gzip -k9v pic{1..5}.jpg

Decompress files with GZIP:

$ gzip --keep --best --verbose file1.gz
$ gzip -k9v pic{1..5}.jpg.gz

Alternatively, use gunzip to decompress:

$ gunzip --keep file1.gz

Note: While uncompressing, the suffix (.gz) is optional if the original filename is not present.

List metadata of a compressed file:

$ gzip --list file.gz

Read the content of a compressed file:

$ gunzip -c file.gz
$ zcat file.gz
$ zless file.gz

BZIP2

Compress files with BZIP2:

$ bzip2 --keep --verbose file

Decompress files with BZIP2:

$ bzip2 --decompress --keep --verbose file.bz2
$ bunzip2 --keep --verbose file2.bz

Read content of a compressed file with BZIP2:

$ bzcat file.bz2
