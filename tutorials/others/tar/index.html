Create Archive

Create a tar archive:

$ tar --create --file archive.tar playground/
$ tar -cf new-archive.tar playground/
$ tar cf another-arch.tar playground/

Note: playground/ or playground will include the playground directory in the *.tar, but playground/* will not.

List the Content of Archive

$ tar --list --file archive.tar
$ tar -tf new-archive.tar
$ tar tf another-arch.tar

Append Files/Folders to Archive

$ tar --append --file archive.tar newfile
$ tar -rf archive.tar newdirContents/*
$ tar rf archive.tar newDir/

Update Archive with Newer Files

$ tar --update --file archive.tar file
$ tar -uf archive.tar contents/*
$ tar uf archive.tar dir

Delete Content from Archive

$ tar --delete --file archive.tar archivedFile
$ tar --delete --file archive.tar archivedContents/*
$ tar --delete --file archive.tar archivedDir

Extract Files

$ tar --extract --file archive.tar
$ tar --get --file archive.tar
$ tar -xf archive.tar
$ tar xf archive.tar

Extract Specific File

$ tar --extract --file archive.tar specifiedFile
$ tar -xf archive.tar --wildcards '*.txt'
$ tar xf archive.tar dir

More Stuff

Create a tgz archive:

$ tar cf hello.tgz *.txt

Create a tbz archive:

$ tar cf hello.bz2 *.py

Use with find:

$ find . -name "*.txt" -exec tar cf text.tar '{}' +

Use with remote server:

$ ssh remote 'tar -cf Documents/' | tar xf -
