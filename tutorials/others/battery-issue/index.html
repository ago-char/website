How to Increase Battery Life in Linux

TLP

TLP is a powerful tool to optimize battery life on Linux. Here's how to install and configure it:

Install TLP

sudo apt install tlp

View TLP Statistics

sudo tlp-stat

Enable TLP-RDW for USB Support

sudo tlp-rdw [enable|disable]

Use TLP as a Service

sudo tlp start

Set TLP to Start at Boot

sudo systemctl enable tlp

Check if TLP Startup Service is Enabled

sudo systemctl is-enabled tlp

By default, Bluetooth may be enabled on startup. To disable it:

sudo nano /etc/tlp.conf

Find the line:

# DEVICES_TO_DISABLE_ON_STARTUP="bluetooth nfc wifi wwan"

Uncomment and modify it to:

DEVICES_TO_DISABLE_ON_STARTUP="bluetooth"

For more options and details, visit TLP Installation Guide.

Powertop

Powertop helps you monitor and tune power consumption. Follow these steps to use it:

Install Powertop

sudo apt install powertop

View Powertop Status

sudo powertop

Start Powertop Service

sudo systemctl start powertop.service

Set Powertop to Start at Boot

sudo systemctl enable powertop.service

Check if Powertop Startup Service is Enabled

sudo systemctl is-enabled powertop
