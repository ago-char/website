üîë Key Terminology Explained

Term Meaning
objectClass A template defining a set of attributes for an LDAP entry. It categorizes entries (e.g., person, organization).
attributeType Defines an individual data field that an LDAP entry can store (e.g., email, common name).
Description A brief explanation of the purpose of the object class or attribute.
SUP "Superior": Indicates inheritance from another objectClass or attributeType.
Type Indicates the nature of the object class (STRUCTURAL, AUXILIARY, or ABSTRACT).
MUST Lists attributes that are required for entries of that object class.
MAY Lists attributes that are optional.
Aliases Alternative names for an attribute.
EQUALITY Specifies how attribute values are compared for equality.
SUBSTR Defines how substring matching should be handled for attribute values.
